== Description ==
This plugin is for <strong>Full Screen Slide-show</strong> and it is in 13kb including libraries, banners and everything. very light and clean code. No dirty business, no ads, no monkey business and no worry for performance.

#**Very easy to set-up.**#
Simply navigate to WordPress administration dashboard >> Settings >> Light Full Screen Slider
#**Very simple to Use.**#
~~~~
Simply add` <?php get_lfs(); ?> `into your theme template
at the place where you want to get sideshow.
~~~~

There are several WordPress plugins available for <strong>slider</strong>, <strong>slide show</strong>. And also  available several plugins for <strong>full screen slide show</strong>. I search for <strong>bloat-less</strong>, <strong>light weight</strong>, fat-less, <u>clean code</u> plugin but surprisingly I fail to find such plugin and I decided to write one for myself and contribute to the WordPress community. 

This plugin is for those people who having fear of WordPress site become bloated with several plugins. Who don't want to get features against performance penalty. This plugin emphasis clean and <strong>light weight code</strong>. Entire plugin size is only 13kb in zip and 23kb in Unzip format including all required libraries and attached image banners.

For instructions of slide show setting please see above <b>"<a href="https://wordpress.org/plugins/light-full-screen-slider/installation/">Installation</a>"</b> menu.
Please see <strong>Slide show</strong> demo at here "<a href="http://itweb.in">Slide Show</a>"
== Installation ==
* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings through WordPress site administration. 
* Navigate *Setting* >> *Light Full Screen Slider*
* On this page add full path including http:// of your images and add caption respectively and save setting. 
* After setup up slide show at setting page you need to add `<?php echo get_lfs(); ?>` into your theme template file.
